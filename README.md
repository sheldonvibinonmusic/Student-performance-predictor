# 🎓 Student Performance Predictor

A beginner Machine Learning project built using Python and Scikit-Learn that predicts student marks based on study hours.

---

## 🚀 Features

✅ Load student performance dataset from CSV

✅ Visualize relationship between study hours and marks

✅ Train a Linear Regression model

✅ Evaluate model performance using:

- Mean Absolute Error (MAE)
- R² Score

✅ Accept custom user input

✅ Predict marks for multiple study-hour values

---

## 🛠 Technologies Used

- Python
- Pandas
- Scikit-Learn
- Matplotlib

---

## 📂 Dataset

The dataset contains:

| Hours Studied | Marks |
|--------------|--------|
| 1 | 15 |
| 2 | 25 |
| 3 | 35 |
| ... | ... |

Stored in:

```text
student_performance.csv
```

---

## 📊 Machine Learning Workflow

1. Load dataset
2. Split data into training and testing sets
3. Train Linear Regression model
4. Evaluate performance
5. Predict marks for user-provided study hours

---

## ▶️ How To Run

Clone repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python main.py
```
